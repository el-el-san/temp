## 🔧 3. 自動化スクリプト（サンプル）

※ 実際に作成する場合は `scripts/append-timestamp.sh` 等のファイル名で配置してください。

```bash
#!/usr/bin/env bash
# scripts/append-timestamp.sh

set -euo pipefail
TARGET=README.md
TS="$(date '+%Y-%m-%d %H:%M:%S')"

# 既存の更新日時行を削除（あれば）
sed -i '/^\*\*更新日時\*\*:/d' "$TARGET"

# 末尾に追記
{
  echo
  echo '---'
  echo "**更新日時**: ${TS}"
} >> "$TARGET"
```
